# KotlinREST
Kotlin REST project

## Libraries and Features
- Navigation
- MVVM architecture
- Retrofit
- Coroutines
- Glide

## Installation

```bash
  run KotlinRESTAPI: https://github.com/darasoylu/KotlinRESTAPI
  change BASE_URL with IPv4 adress
```
